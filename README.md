# Real-Time Object Detection using YOLOv8

## Description
This project implements a real-time object detection pipeline using a pre-trained YOLOv8 Nano model. It captures video from a webcam, detects objects, draws bounding boxes, logs detections, and measures FPS performance.

## Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- NumPy
- Pandas

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python object_detection.py

3. Press 'Q' to exit.

## Outputs
- detections.csv: Detection logs
- performance_report.txt: FPS and statistics
